Core Components
---------------

The SDK is organized around a central :class:`SegmindClient` that provides access to various service modules:

* :doc:`client` - Main client class for API interactions
* :doc:`exceptions` - Exception classes for error handling
* :doc:`models` - Model discovery and information
* :doc:`pixelflows` - Workflow automation
* :doc:`webhooks` - Event notifications
* :doc:`files` - File upload and management
* :doc:`generations` - Generation history
* :doc:`accounts` - Account management
